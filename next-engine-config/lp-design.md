## ビジネスモデル

### サービス概要

**サービス名**: ネクストエンジン設定代行サービス

**キャッチコピー候補**:
1. "ネクストエンジンの設定、3日で完了。あなたは販売に集中できます。"
2. "マルチモール運営を自動化。ネクストエンジン設定のプロにお任せください。"
3. "EC運営の効率化を、今すぐ。ネクストエンジン導入から運用まで完全サポート。"

**バリュープロポジション**:
- 複雑なネクストエンジン設定を3日で完了
- API連携・トークン自動更新で運用コスト削減
- マルチモール（楽天・Amazon・Yahoo・Qoo10）統合管理
- 月1000回のAPI無料枠を最大活用（課金防止）

### 提供サービス

#### 1. 初期セットアップパック（¥80,000）

**含まれる内容**:
- ネクストエンジンアカウント設定支援
- OAuth認証設定（トークン取得）
- API接続設定
- 基本設定ファイル作成（商品マスタ、在庫、配送）

**納期**: 3営業日

**成果物**:
- 設定済みネクストエンジンアカウント
- 設定ドキュメント（Markdown形式）
- 操作マニュアル

#### 2. 商品マスタ統合パック（¥50,000〜）

**含まれる内容**:
- 既存商品データのエクスポート
- マルチモール商品コード統合設計
- 商品マスタインポート設定
- 差分同期設定（既存商品考慮）

**対応モール**:
- 楽天市場
- Amazon（自社出荷・FBA）
- Yahoo!ショッピング
- Qoo10

**納期**: 5営業日〜（商品数による）

#### 3. 在庫管理自動化パック（¥40,000）

**含まれる内容**:
- 在庫連携設定（統合在庫 or モール別在庫）
- 在庫アラート設定
- 倉庫別在庫管理設定（複数倉庫対応）
- 在庫同期スケジュール設定

**納期**: 3営業日

#### 4. API自動化パック（¥60,000）

**含まれる内容**:
- Cloudflare Workers デプロイ
- トークン自動更新設定（1日2回Cron）
- API使用量管理設定（90%自動ブロック）
- D1データベース設定
- 使用量モニタリングダッシュボード

**納期**: 5営業日

**月額料金**: なし（Cloudflare無料枠内）

#### 5. フルサポートプラン（¥150,000 + 月額¥30,000）

**初期設定（¥150,000）**:
- 上記1〜4すべて含む
- カスタム設定（業種特有の要件に対応）
- データ移行支援

**月額サポート（¥30,000）**:
- 設定変更・調整（月3回まで）
- API使用状況レポート（月次）
- トラブルシューティング
- 新機能追加サポート
- Slack/メールサポート（平日10:00-18:00）

**納期**: 10営業日

#### 6. スポット対応（時間単価）

**料金**: ¥15,000/時間

**用途**:
- 緊急トラブル対応
- 単発の設定変更
- コンサルティング

## 技術スタック

### フロントエンド

**HTML/CSS/JavaScript**:
- 理由: シンプルで高速、Cloudflare Pagesと相性が良い
- フレームワーク不要（静的サイト）

**CSSフレームワーク**:
- **Tailwind CSS** (推奨)
  - ユーティリティファーストで開発速度が速い
  - レスポンシブ対応が容易
  - ファイルサイズが小さい

**アイコン**:
- **Heroicons** (Tailwind公式)
- **Font Awesome** (代替案)

**アニメーション**:
- **AOS (Animate On Scroll)**
  - スクロールアニメーションで視覚的魅力を追加
  - 軽量（< 10KB）

### バックエンド（お問い合わせフォーム）

**オプション1: Cloudflare Workers + D1 (推奨)**

メリット:
- Cloudflare無料枠内で運用可能
- サーバーレスで管理不要
- 既存のAPI使用量管理Workerと統合可能

実装:
```
POST /api/contact
├─ バリデーション
├─ D1に保存
├─ メール通知（Mailchannels）
└─ サンクスページへリダイレクト
```

**オプション2: Cloudflare Forms**

メリット:
- 最もシンプル
- 設定不要

デメリット:
- カスタマイズ性が低い

**オプション3: Google Forms埋め込み**

メリット:
- 無料
- スプレッドシート連携

デメリット:
- デザインの統一感が損なわれる

### ホスティング

**Cloudflare Pages**

メリット:
- 無料（500ビルド/月、帯域無制限）
- 自動デプロイ（Git連携）
- カスタムドメイン無料
- CDN標準装備（高速）
- SSL/TLS証明書自動発行

設定:
```yaml
# wrangler.toml
name = "nextengine-lp"
compatibility_date = "2024-01-01"

[build]
command = "npm run build"
output_directory = "dist"

[[pages.redirects]]
from = "/contact/success"
to = "/thank-you.html"
status = 301
```

### アナリティクス

**Google Analytics 4**:
- 無料
- 詳細なユーザー行動分析
- コンバージョン追跡

**Cloudflare Web Analytics** (代替案):
- プライバシー重視
- Cookie不要
- Cloudflare管理画面で確認可能

### メール送信

**Mailchannels (Cloudflare Workers経由)**:
- 無料枠: 月10,000通
- トランザクションメールに最適
- SPF/DKIM設定可能

### ドメイン

**推奨ドメイン名**:
- `nextengine-setup.com`
- `ne-support.jp`
- `ec-automation.jp`

**取得先**:
- Cloudflare Registrar（最安値）
- お名前.com
- Google Domains
